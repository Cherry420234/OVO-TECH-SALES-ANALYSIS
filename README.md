# OVO-TECH-SALES-ANALYSIS
Interactive Microsoft Excel Sales Dashboard analyzing 2,098 transactions using Pivot Tables, Charts, KPIs, and Slicers to deliver actionable business insights.
# PROJECT OVERVIEW
Businesses generate large volumes of sales data, but transforming that data into actionable insights is often challenging. This project demonstrates how Microsoft Excel can be used to clean, analyze, and visualize sales data through an interactive dashboard. Using Pivot Tables, Pivot Charts, KPIs, and Slicers, the dashboard enables stakeholders to monitor business performance, identify sales trends, evaluate product profitability, and compare regional and sales representative performance.
# PROBLEM STATEMENT
Management requires a centralized reporting dashboard that answers key business questions such as:
1.  Which products generate the highest revenue and profit?
2.  Which regions contribute the most to company performance?
3.  Who are the top-performing sales representatives?
4.  How do sales fluctuate over time?
5.  What is the company's overall profitability?
